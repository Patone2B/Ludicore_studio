À propos

Un espace pensé pour créer sans se disperser.

Ludicore Studio est un espace de création destiné aux personnes qui veulent transformer une idée de jeu en prototype clair. L’objectif est de regrouper les étapes importantes : cartes, plateau, règles, composants, ambiance sonore, quêtes et personnages.

Le studio reste volontairement simple : chaque module correspond à une tâche précise, afin de pouvoir avancer rapidement, tester son jeu, corriger les règles et améliorer le projet sans perdre le fil.

Licence GPL (GNU General Public License).

Ce logiciel est distribué sous la licence GPL, une licence open source copyleft qui garantit les libertés suivantes :

  • Utilisation : libre pour tout usage, y compris commercial.
  • Modification : vous pouvez adapter le code à vos besoins.
  • Redistribution : vous pouvez partager le logiciel, à condition que le code source reste accessible, que toute redistribution conserve la même licence GPL et que les mentions d’origine soient préservées.

Contrainte importante : il n’est pas possible d’intégrer ce code dans un projet propriétaire ou fermé sans publier son code source. Cette licence est idéale pour les projets qui veulent protéger la liberté du logiciel sur le long terme.

À quoi sert ce logiciel ?

Ce logiciel sert à créer ce dont vous avez besoin pour mettre en forme une idée de jeu de société, mais aussi pour noter et organiser des idées de quêtes pour de futurs jeux de rôle.

Il comporte plusieurs modules : création de cartes de jeu, création de map, rédaction des règles, création de pions et jetons, bande son, quêtes et fiche personnage. Il peut servir pour une esquisse, un prototype, ou pourquoi pas pour une utilisation commerciale.

Par quel module commencer ?

Le plus simple est de commencer par les règles ou les cartes. Les règles donnent la structure du jeu, tandis que les cartes permettent de tester rapidement les idées principales.
