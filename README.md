# 🎲 Ludicore Studio

**Ludicore Studio** est un logiciel open source dédié à la création de jeux de plateau et de règles personnalisées.  
Il permet de concevoir facilement des contenus ludiques sans passer par des outils complexes comme Word ou InDesign.

---

## 🚀 Pourquoi utiliser Ludicore Studio ?

Créer un jeu de plateau peut vite devenir compliqué :
- mise en page difficile
- règles peu lisibles
- outils non adaptés

👉 Ludicore Studio apporte une solution simple :

- 🧩 Création rapide de règles de jeu
- 📝 Interface pensée pour les créateurs
- 📄 Export propre et lisible
- 💾 Sauvegarde en fichiers JSON (faciles à modifier et partager)

## 🛡️ Sécurité

Ludicore Studio adopte une approche transparente et respectueuse de vos données :

- ✔️ Aucune collecte de données  
- ✔️ Aucun envoi vers un serveur  
- ✔️ Fonctionnement 100% local  
- ✔️ Données stockées sur votre machine uniquement  

🔍 Analyse de sécurité :
- Application testée avec plus de 60 antivirus (VirusTotal)
- Aucun malware détecté sur le fichier principal
- Une détection isolée peut apparaître sur un composant d’installation (faux positif courant)

⚠️ Note :
Le logiciel n’est pas signé numériquement, ce qui peut afficher un avertissement Windows (“éditeur inconnu”).  
Cela est normal pour un projet open source indépendant.
