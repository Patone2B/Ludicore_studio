## 📜 **Droits et libertés**
En utilisant ce logiciel, vous bénéficiez des **quatre libertés fondamentales** définies par la Free Software Foundation :

1. **Liberté d’utilisation** :
   - Vous pouvez exécuter le programme pour **n’importe quel usage** (personnel, commercial, éducatif, etc.).

2. **Liberté d’étude et de modification** :
   - Vous avez accès au **code source complet** et pouvez l’adapter à vos besoins.

3. **Liberté de redistribution** :
   - Vous pouvez partager des copies du logiciel **gratuitement ou contre rémunération**.

4. **Liberté de distribution des versions modifiées** :
   - Vous pouvez distribuer vos propres versions modifiées, **à condition** de :
     - Publier leur code source sous **la même licence GPLv3**.
     - Conserver les **mentions d’origine** (auteur, nom du projet, licence).

Pour respecter la GPLv3, vous devez :

Nous avons choisi cette licence pour :
